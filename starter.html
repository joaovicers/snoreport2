Introduction

SnoReport 2.0 is a tool that identifies the two main classes of snoRNAs, using a combination of secondary structure prediction and machine learning. In contrast to previous methods for snoRNA identification (except snoSeeker), snoReport prediction does not use information of putative target sites within ribosomal or spliceosomal RNA. However, many orphan snoRNAs have been discovered with the snoReport 1.0 approach. The target(s) of orphan snoRNAs are not known, consequently such genes would be missed by target depending on the identification method. Beyond this, some snoRNAs are shown to target specific mRNAs, suggesting other functions, e.g., interference with A-to-I editing.

Here, we present an update of snoReport, by extracting new features for both C/D and H/ACA box snoRNAs, developing a more sophisticated technique in the SVM training phase (with recent data from vertebrate organisms and a different approach to refine the C and Î³ SVM parameters), and using new versions of the tools and databases previously taken to build snoReport. To validate this new version of snoReport, we tested it in different organisms. These experiments have shown a very good performance.

Howto

USAGE:

snoreport_2 -i <fasta_file> [-CD|-HACA] [-trainCD|-trainHACA] [OPTIONS]

-h,--help
Print help and exit.
--verbose
Verbose mode.

General Options

-i <fasta file>
Inform the input file (fasta file).
-CD
Predict C/D box snoRNAs.
-HACA
Predict H/ACA box snoRNA.
--trainCD, --TrainHACA
Generate feature vectors from snoRNA candidates (for machine learning tasks only).
--positives
Output only positive predictions.
-o <file>>
Generate a fasta file with the putative snoRNAs found.
--PS
Creates a postScript visualization of the predicted snoRNAs

snoReport have two types of output:

  • Fasta file output: generate a fasta file with the predicted snoRNAS
  • <STDOUT≶ output: Generate a table informing:

For C/D box:

#Seq_ID #Start #End #C box #C box start position #D box #D box start pos #SVM prob #veredict

e.g

seq_1 1 142 TTGATGA 11 CTGA 133 0.96 C/D box snoRNA

For H/ACA box is the same, but C and D boxes are replaced by H and ACA box, respectively.
